# Bank Loan Analysis using Python
End-to-End Bank Loan Analysis using Python, Pandas, Matplotlib and Plotly.

## 📌 Project Overview
This project analyzes bank loan data using Python to identify trends in loan applications, funded amounts, repayments, loan purposes, home ownership, employee length, and regional performance.

## 📊 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Plotly Express
- Jupyter Notebook

## 📈 KPIs
- Total Loan Applications
- Total Funded Amount
- Total Amount Received
- Average Interest Rate
- Average DTI

## 📉 Visualizations
- Monthly Loan Applications
- Monthly Funded Amount
- Monthly Amount Received
- Regional Analysis by State
- Loan Purpose Analysis
- Employee Length Analysis
- Home Ownership Treemap
- Loan Term Doughnut Chart
